# StudyFlow 测试说明

## 测试环境

- **操作系统**: HarmonyOS NEXT
- **测试方式**: Mock 模式手动测试
- **测试数据**: 内置示例（概率论与数理统计、数据结构、大学英语）

## v3.0 新增测试用例

### TC-21: 考试安排导入与倒计时

| 项目 | 内容 |
|------|------|
| 测试目标 | 验证粘贴考试安排文本后生成倒计时 |
| 前置条件 | 在主页面 |
| 操作步骤 | 1. 点击考试倒计时卡片<br>2. 点击"添加"按钮<br>3. 粘贴考试安排文本<br>4. 点击"确认添加" |
| 预期结果 | 1. 显示已识别 N 门考试<br>2. 考试列表按优先级排序<br>3. 最近考试显示倒计时天数<br>4. 高优先级显示"临近"标签 |

### TC-22: 复习题导入与卡片拆分

| 项目 | 内容 |
|------|------|
| 测试目标 | 验证粘贴复习题文本后拆分为独立卡片 |
| 前置条件 | 在主页面 |
| 操作步骤 | 1. 点击"加载示例"加载示例复习题<br>2. 输入课程名称<br>3. 点击"开始解析"<br>4. 查看解析结果 |
| 预期结果 | 1. 显示已识别 N 道题<br>2. 每道题显示题目摘要和难度<br>3. 显示"本地解析"或"智能解析"标签<br>4. "开始刷题"按钮可点击 |

### TC-23: 沉浸式卡片复习

| 项目 | 内容 |
|------|------|
| 测试目标 | 验证单卡片复习、查看答案、反馈完整流程 |
| 前置条件 | 已导入题目或从首页点击"开始今日复习" |
| 操作步骤 | 1. 进入 ImmersiveReviewPage<br>2. 阅读题目<br>3. 点击"查看答案"<br>4. 查看答案和解析<br>5. 点击"认识"/"模糊"/"忘了" |
| 预期结果 | 1. 进度条显示当前进度<br>2. 答案区域在点击后显示<br>3. 解析和常见错误可见<br>4. 点击反馈后自动进入下一张<br>5. 完成所有卡片后显示统计 |

### TC-24: 错题自动收集

| 项目 | 内容 |
|------|------|
| 测试目标 | 验证"忘了"反馈的题目自动加入错题本 |
| 前置条件 | 在沉浸式复习中 |
| 操作步骤 | 1. 对某道题选择"忘了"<br>2. 完成复习后进入错题本 |
| 预期结果 | 1. "忘了"的题目出现在错题本中<br>2. 显示题目、用户答案状态<br>3. 可标记"掌握了" |

### TC-25: AI 状态检测

| 项目 | 内容 |
|------|------|
| 测试目标 | 验证设置页 AI 状态面板正常工作 |
| 前置条件 | 在"我的"→"学习设置" |
| 操作步骤 | 1. 查看 AI 状态区域<br>2. 点击"测试连接"<br>3. 观察状态变化 |
| 预期结果 | 1. 显示当前模式（Mock/在线）<br>2. 显示 API Key 配置状态<br>3. 测试连接后显示结果 |

### TC-26: 完整闭环（v3.0）

| 项目 | 内容 |
|------|------|
| 测试目标 | 验证从导入到复习的完整流程 |
| 前置条件 | 冷启动应用 |
| 操作步骤 | 1. 游客登录<br>2. 导入考试安排 → 生成倒计时<br>3. 导入复习题 → 拆分卡片<br>4. 进入沉浸式复习<br>5. 完成反馈（混合选择）<br>6. 查看错题本 |
| 预期结果 | 1. 全流程无崩溃<br>2. 数据在页面间保持一致<br>3. 错题本实时更新<br>4. 首页统计数据正确 |
