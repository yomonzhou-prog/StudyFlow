# StudyFlow OS 测试说明

## 测试环境

- **操作系统**: HarmonyOS NEXT
- **测试方式**: Mock 模式手动测试（不依赖 DeepSeek API）
- **测试数据**: 内置三套示例课程（概率论与数理统计、数据结构、大学英语）
