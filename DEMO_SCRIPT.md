# StudyFlow 答辩演示脚本

## 演示环境准备

- DevEco Studio 模拟器（HarmonyOS NEXT）
- Mock 模式运行（无需网络）
- 预置示例数据：概率论与数理统计、数据结构、大学英语

## 演示流程

### 1. 游客登录（30秒）

1. 启动应用 → SplashPage 显示 "StudyFlow — 期末复习，从一张卡片开始"
2. 自动跳转 OnboardingPage（3页引导）
3. 点击"开始学习"进入 LoginPage
4. 点击"游客体验"直接进入 MainPage
5. **要点**：强调简约设计，无 emoji 无 AI 横幅

### 2. 期末复习首页（30秒）

1. 顶部显示问候语和状态行
2. 考试倒计时卡片：概率论与数理统计，还剩 5 天
3. 今日复习入口：N 张卡片 · 约 X 分钟
4. 复习优先级 Top 3：方差、协方差、最大似然估计
5. 底部轻量统计：已复习 / 错题 / 连续天数
6. **要点**：不是任务列表，是"今天该复习什么"

### 3. 导入考试安排（30秒）

1. 点击考试倒计时卡片 → ExamSchedulePage
2. 显示 3 门考试倒计时
3. 点击"添加" → 粘贴考试安排文本
4. 自动解析为结构化考试列表
5. **要点**：OCR 功能开发中，当前支持文本化输入

### 4. 导入复习题（1分钟）

1. 返回首页，点击"开始今日复习" → ImmersiveReviewPage
2. 如无卡片 → 点击"导入题目" → ExamPaperImportPage
3. 点击"加载示例" → 自动填充 5 道概率论题目
4. 点击"开始解析" → 解析完成
5. 显示已识别 5 道题，点击"开始刷题"

### 5. 沉浸式复习（1分钟）

1. 全屏单卡片显示第一道题
2. 点击"查看答案" → 显示答案和解析
3. 对第1题点击"认识"（正确）
4. 自动进入第2题
5. 点击"查看答案"，选择"模糊"
6. 第3题选择"忘了"

### 6. 复习完成与错题（30秒）

1. 完成所有卡片 → 显示统计（认识 X / 模糊 X / 忘了 X）
2. X 道错题已加入错题本
3. 返回 → 进入"我的" → "错题本"
4. 显示刚才标记为"忘了"的题目
5. 演示"掌握了"标记功能

### 7. 学习教练（30秒）

1. 进入"教练" Tab
2. 显示快捷提问卡片
3. 点击"我的薄弱点有哪些"
4. 显示教练回复（本地规则分析）
5. **要点**：不是聊天机器人，是学习策略建议

### 8. AI 状态检测（15秒）

1. 进入"我的" → "学习设置"
2. 滚动到 AI 状态区域
3. 显示 "Mock 模式" 或 "在线模式"
4. 点击"测试连接"
5. **要点**：明确显示当前数据模式

## 演示要点

- 全程 Mock 模式，不依赖网络
- 强调从"任务列表"到"复习卡片"的体验转变
- 强调简约高级的产品风格
- 强调沉浸式单卡片复习体验
- 强调复习优先级算法（老师重点 + 考试临近 + 错题反馈 + 难度）
